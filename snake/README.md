## Игра классическая змейка на python с отладочным меню

Данная змейка реализована на фреймворке pygame с использование ООП

Релизованы классы:
- **Snake:**
    - Инициализация
    - Обновление:
        - Создание корректной длины змейки
        - Функция смерти
- **Apple:**
    - Инициализация
    - Обновление:
        - Создание клетки яблока

***
Релизованы функции:
- **draw_grid:**
    - Создание клеток
- **main:**
    - Отрисовка:
        - Snake
        - Apple
        - draw_grid
        - Текущих данных игры:
            - Количество очков
            - Длина змеи
            - Кординаты головы змеи
            - Напровление головы
    - Обновление
